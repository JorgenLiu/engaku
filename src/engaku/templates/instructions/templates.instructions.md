Template file conventions for this project.

Template files are copied verbatim to target repos by `engaku init`. When updating any agent or hook template, always update BOTH the template version (`src/.../templates/`) AND the live version (`.github/`) in the same operation. Keep templates as generic stubs — projects fill in their own conventions using `.github/instructions/` files after init.
